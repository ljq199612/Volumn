<h3 class = 'auto-sort-sub'> 消息(数据)的发布与订阅</h3>


<div class="myTip">

数据（消息）的发送者（发布者）不会直接把消息发送给接收者。发布者以某种方式对消息进行分类，接收者（订阅者）接收特定类型的消息。 发布与订阅系统一般会有一个发布消息的中心点`broker`。
</div>
